在 Centos 中安装 Java8

最后更新: 2019-10-08 13:45:03 字数: 117 阅读时间: 1 min 上一页 首页

前言

由于要在服务器上安装 Jenkins 所以首先要安装 java。

更新

安装之前最好更新下

yum -y update

安装 Java8

yum install java-1.8.0-openjdk

查看版本

java -version

设置环境变量 JAVA_HOME

首先找到安装 java 的位置

update-alternatives --config java

编辑 .bash_profile 文件

vim .bash_profile

添加这一行

export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.191.b12-1.el7_6.x86_64/jre/bin/java

重新执行刚修改的 .bash_profile

source .bash_profile

验证一下

echo $JAVA_HOME
